# 聊天记录 — 2026-05-13

## 主题: 视觉定位瞄准系统方案设计与代码实现

### 需求
- 构建视觉定位 + 鼠标键盘操作系统，用于 CS:GO/Valorant/FPS 游戏
- 使用 OBS 获取画面 → 后台推理 → PID 控制鼠标 → 准星定位到目标
- 推理用 TensorRT，运行在 Windows 平台
- 强化学习数据采集

**测试:** 18 个测试全部通过 (pytest)

### 反作弊核心设计 (Humanizer)
1. 感知延迟: 150-250ms 对数正态分布
2. 位置噪声: σ=3-5px
3. 追踪不完全: 只跟到 80-90%
4. 过冲概率: 15-20% (超过目标再拉回)
5. 微抖动: 1.5-2px @ 3-5Hz sin 波
6. 间歇暂停: 每 30-90s 随机暂停 2-5s
7. 游戏差异化: CS=0.5, Valorant=0.3 (更保守)
