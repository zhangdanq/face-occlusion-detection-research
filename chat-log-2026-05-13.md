- 配置代理: 创建 .env 文件 (UV_HTTP_PROXY/UV_HTTPS_PROXY)
- 创建完整项目结构:
  ```
  src/visual_targeting/
  ├── system.py              # CLI 入口 + 主循环
  ├── capture/
  │   ├── __init__.py         # 抽象接口 + 工厂函数
  │   ├── mock_capture.py     # macOS: webcam/video/test
  │   └── dxgi_capture.py    # Windows: DXGI
  ├── inference/
  │   ├── __init__.py         # 抽象接口 + 工厂函数
  │   ├── mock_detector.py    # macOS: ultralytics CPU/MPS
  │   ├── trt_detector.py    # Windows: TensorRT
  │   └── model_export.py    # 模型导出工具
  └── control/
      ├── pid.py              # PID (速度形式, 无dt除法)
      ├── humanizer.py        # 人化模块
      ├── mapper.py           # 坐标映射
      ├── mouse_input.py      # SendInput (Windows only)
      └── target_selector.py  # 目标选择
  ```
- 编写 18 个测试 (pytest): PID/Humanizer/Mapper/TargetSelector
